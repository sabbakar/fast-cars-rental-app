# Fast Cars - Car Rental System

A car rental web application built with Flask and MySQL.

## Features

- User Registration & Login  
- Car Listings & Booking  
- Admin Dashboard  
- Testimonials  
- Contact Page
- Booking page

## Requirements

- Python 3  
- Flask  
- MySQL  
- XAMPP (or other MySQL setup)

## Setup Instructions

1. **Clone the repository**
   ```bash

   cd fast-cars
